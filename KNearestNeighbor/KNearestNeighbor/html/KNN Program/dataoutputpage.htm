Data Output Page

Function: The purpose of the page is to display the output data.

Upon entering the Data Output Page, it will be mostly blank (See Figure 1). You will notice, however, that it does provide you with the output class name in the View Data Output panel.

If you want to try new information, you may (at any time) click the "Back" button or the "New" button. The "Back" button will take you back to the Data Input Page with all of your information still entered whereas the "New" button will start the program from the Start Page.

I will be referring to the panels throughout this document. The panels are simply the titles of the different sections on the page. So for instance, on the Data Output Page (step 3), there are 3 panels. There is the View Data Output Panel, Plot's Closest Competitor Panel, and Graph Panel. When I refer to these panels I refer to their contents as well.

Recommended Steps:

  1. Click the dropdown next to "Plot X:" and choose an attribute to view. Note: This choice must be different from the one for "Plot Y".
  2. Click the dropdown next to "Plot Y:" and choose an attribute to view. Note: This choice must be different from the one for "Plot X".
  3. Determine if you would like to view the distances computed from the input data point to the training data points.
  4. Click Plot.

Now we will be looking at each of these sections in a bit closer detail.

k-NN Closest Competitor Panel

This panel will show the k-nearest neighbor's class to your input given the k value. In other words, it displays the classification of the input data point.

Graph Panel

This panel will first appear empty when you first visit the step. It will have the classes of your training data off to the right in the legend however.

First you must select an x-coordinate and a y-coordinate to plot. This will compute the location of the data given just the two attributes that you choose. It will also produce the nearest neighbor given ONLY those two attributes. This won't necessarily reflect the output class in the k-NN Closest Competitor Panel. For the k-NN algorithm, it considers all of the dimensions not just 2-dimensions.

After selecting the two attributes to plot, you may choose to view the distances of the k-nearest points. This way it will give you an idea as to what else was around the data point and how far it really was. You may also choose to view all of the distances as well.

After making your selections, click "Plot" and the graph will populate with the data points including your data point. If you chose either of the distance options, while hovering you will be able to see the distances depending on which one you chose.

In addition, you will notice that the Plot's Closest Competitor Panel has also been updated. You may choose differnet points to plot and choose plot again as well.

To view more on the Plot's Closest Competitor, view the next section.

Plot's Closest Competitor

This panel contains the k-NN calculated with the k value from the Data Input Page and the attributes chosen in the Graph Panel under the "Plot X" and "Plot Y" dropdowns.

You'll notice at first the data is marked N/A. This is because you haven't assigned attributes to the algorithm to compute.

After choosing data to plot (see Graph Panel), you will see that the information has been updated.

This shows the closest competitor's data label, it's class, and its distance from the input point. This is helpful to get a sort of general idea as to how the different attributes affected the ultimate outcome of the input's overall class.
