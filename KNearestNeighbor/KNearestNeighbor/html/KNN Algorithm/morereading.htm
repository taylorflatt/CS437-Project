More on k-NN

Although this help file attempts to be as comprehensive (thereby inclusive) as possible, there are just some things that are good sources that should be mentioned for further reading.

  • (Recommended) Tinoosh Mohsenin. This is probably the most exhaustive look at k-NN there is. The pdf looks at every facet of the algorithm and a systematic approach on how to improve the accuracy of the algorithm.
  • CourseA Machine Learning. This is a good source of videos by a fantastic lecturer. Among other things he talks in detail about the k-NN algorithm in the week four "instance-based learning".
  • Analytics Vidhya. This is a nice gentle introduction to the k-NN algorithm with a brief overview of its function and attributes.
  • Stat Soft. They provide an overview of the k-NN algorithm much like Analytics Vidhya but it is much more detailed and include a much more mathematical approach to the algorithm.
  • Saed Sayad. There is a general look at k-NN with a good example and explanation of handling discrete measurements with Hamming Distance. Although, it should be noted that VDM is a better choice usually.
